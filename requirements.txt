# ============================================
# 開發者安裝步驟:
# ============================================
#
# 1. 安裝 Python 套件
#    pip install -r requirements.txt
#
# 2. 安裝 Playwright 瀏覽器
#    playwright install chromium
#
# 3. 準備知識庫檔案
#    將 .csv, .xlsx, .txt, .pdf 放入 knowledge_base/ 資料夾
#
# 4. 複製環境變數檔案
#    copy .env.example .env  (Windows)
#    cp .env.example .env    (Mac/Linux)
#
# 5. 編輯 .env 填入您的 Gemini API Key
#    取得方式: https://aistudio.google.com/apikey
#
# 6. 啟動 GUI 啟動器
#    python launcher.py
#
# ============================================
# 打包成 EXE (商業發布):
# ============================================
#
# 1. 安裝打包依賴
#    pip install -r requirements_build.txt
#
# 2. 執行打包腳本
#    python build.py
#
# 3. 輸出檔案
#    dist/蝦皮AI客服機器人/蝦皮AI客服機器人.exe
#
# 4. 發布
#    將 dist/蝦皮AI客服機器人/ 資料夾發給客戶
#    客戶雙擊 .exe 即可使用，無需安裝 Python
